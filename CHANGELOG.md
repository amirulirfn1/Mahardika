- Initial alpha release of Mahardika monorepo
- UI component library (`@mahardika/ui`): Button, Card, BrandButton, BrandCard, AIChat, etc.
- Next.js web application (`@mahardika/web`)
- Sentry error monitoring setup
- GitHub Actions CI (Node 18/20 with pnpm cache)
- Husky pre-push hooks, ESLint, Prettier, Vitest
- License (MIT) and contributing guidelines
